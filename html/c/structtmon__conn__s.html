Detailed Description

tmon connection monitoring object. Created by tmon_conn_create(). Create one per receiver delivery controller from inside the source_notification_function (receiver) callback. That callback can return this object as the source-specific clientd. See Client Data Pointers. Application must not write to fields in this structure.

Methods: tmon_conn_create(), tmon_conn_delete(), tmon_conn_rcv_event().
