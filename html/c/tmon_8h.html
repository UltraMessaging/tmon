time_t  cur_time

Portable, reentrant converter from Posix "time_t" to ascii date/time.

Unix "time_t" is number of seconds since the Unix Epoch (midnight, 1-Jan-1970).

Parameters
bufferByte array at least 27 bytes long. Date/time string will be written here.
bufsizeActual number of bytes in "buffer".
cur_timePosix time value to convert.

◆ tmon_usleep()

void tmon_usleep ( int  usec)

Portable microsecond-precision sleep function.

Note that there is no guarantee of accuracy. Many operating systems do not support sleep times under a millisecond. User of this function should be prepared to experience much longer sleep times than requested.

Parameters
usecnumber of microseconds to sleep.
